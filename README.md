# 酒店信息展示系统

这是一个专为技术小白设计的酒店信息展示系统，通过GitHub Pages实现，无需编程基础即可维护和部署。

## 📋 项目功能

- 酒店列表展示，支持按大洲和国家筛选
- 酒店详情查看，包括价格、地址、描述和可用情况
- 按更新日期自动排序（最新更新的酒店显示在前面）
- 深色/浅色主题切换
- 响应式设计，适配手机和电脑

## 🔧 前提条件

在开始之前，您需要准备以下内容：

1. **GitHub账号**：如果没有，请先在[GitHub官网](https://github.com)注册
2. **基础的电脑操作能力**：如复制、粘贴文件等

## 🚀 如何创建GitHub仓库

### 步骤1：注册/登录GitHub

1. 访问[GitHub官网](https://github.com)
2. 点击右上角的"Sign up"注册账号，或点击"Sign in"登录已有账号

### 步骤2：创建新仓库

1. 登录后，点击右上角的"+"号，选择"New repository"
2. 在"Repository name"中输入仓库名称（如：hotel-info展示系统）
3. 在"Description"中输入简短描述（可选）
4. 选择仓库可见性：建议选择"Public"（公开）
5. 勾选"Add a README file"选项
6. 点击底部的"Create repository"按钮

### 步骤3：上传项目文件

创建好仓库后，您需要将项目文件上传到GitHub：

1. 在仓库页面点击"Add file"按钮，选择"Upload files"
2. 点击"choose your files"或拖拽文件到虚线框内，上传本项目的所有文件
3. 滚动到页面底部，在"Commit changes"区域输入提交信息（如："Initial commit"）
4. 点击"Commit changes"按钮完成上传

## 🛠️ 如何部署网站

GitHub提供了免费的Pages服务，可以将您的项目部署为网站：

1. 在仓库页面点击顶部的"Settings"选项卡
2. 在左侧菜单中选择"Pages"
3. 在"Source"部分，将"None"更改为"main"分支
4. 点击"Save"按钮
5. 等待几分钟后，页面会显示您的网站地址（通常为：`https://[您的用户名].github.io/[仓库名称]`）

## 📝 如何更新酒店信息

作为"后台"更新酒店信息，您只需要修改`src/pages/Home.tsx`文件中的酒店数据：

1. 访问您的GitHub仓库
2. 点击`src`文件夹，然后点击`pages`文件夹，最后点击`Home.tsx`文件
3. 点击右上角的铅笔图标（Edit this file）
4. 找到`const hotels: Hotel[] = [`部分，这是酒店数据列表
5. 您可以：
   - **添加新酒店**：按照现有格式添加新的酒店对象
   - **修改酒店信息**：直接编辑现有酒店的数据
   - **删除酒店**：删除对应的酒店对象
6. 更新完成后，滚动到页面底部，输入提交信息，然后点击"Commit changes"

> **提示**：每次更新后，网站的"最后更新"日期会自动更新

## 🌍 酒店数据格式说明

每个酒店对象包含以下字段：

```javascript
{
  id: 数字,           // 唯一标识
  name: "酒店名称",    // 酒店名称
  description: "描述", // 酒店描述
  address: "地址",     // 酒店地址
  price: 数字,         // 每晚价格
  image: "图片URL",    // 酒店图片地址
  availability: [      // 可用日期信息
    { date: "2025-12-01", status: "available" },
    { date: "2025-12-02", status: "booked" }
  ],
  continent: "大洲",   // 所属大洲
  country: "国家",     // 所属国家
  updatedAt: "2025-11-25" // 更新日期
}
```

## ❓ 常见问题

### 1. 我可以使用自己的图片吗？

可以。您可以将图片上传到GitHub仓库，然后使用相对路径，或者使用其他图片托管服务的URL。

### 2. 网站加载速度慢怎么办？

如果页面包含大量图片，可能会影响加载速度。建议使用压缩后的图片，或者减少图片数量。

### 3. 如何修改网站标题？

您可以编辑`index.html`文件中的`<title>`标签内容。

### 4. 我可以添加更多筛选条件吗？

目前系统仅支持按大洲和国家筛选。如果需要更多功能，可能需要学习一些基础的编程知识。
